readPhantom: Voxel phantom data processing for PENELOPE simulations

The readPhantom script is a powerful tool designed to streamline/optimize the workflow for researchers and students working with voxel phantoms. It automates the conversion of raw voxel phantom data into structured .vox files suitable for simulation and visualization with the PENELOPE/penEasy Monte Carlo (MC) framework.

This script was published in a peer-reviewed article in Radiation Physics and Chemistry (doi:10.1016/j.radphyschem.2019.02.014) and has been utilized in multiple research projects, contributing to advancements in medical physics and radiation dosimetry.

Keywords: Data Wrangling, ETL, python, pandas, numpy, Fortran, Voxel phantoms, Monte Carlo simulations
